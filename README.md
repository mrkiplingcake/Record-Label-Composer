# 🎵 Record Label Composer

A Windows desktop application for designing and printing precision
100 mm vinyl record centre labels.

Designed with real-world lathe-cut vinyl production in mind.

## Current Status

🚧 Under active development

Current Release

0.2 — Workbench

Next Release

0.3 — Artwork

## Planned Features

- Load artwork
- Drag and position images
- Zoom
- Crop
- Four-up A4 PDF export
- Precision cutter guides
- Project save/load
